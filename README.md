# 📚 SQL Project: Online Bookstore

A SQL Server (T-SQL) project analyzing an online bookstore database.

## 🗂️ Database Schema
| Table | Key Columns |
|-------|------------|
| Books | Book_ID, Title, Author, Genre, Price, Stock |
| Customers | Customer_ID, Name, City, Country |
| Orders | Order_ID, Customer_ID, Book_ID, Quantity, Total_Amount |

## 📋 Queries Covered
**Basic (11):** Filtering, aggregation, sorting, subqueries  
**Advanced (9):** JOINs, GROUP BY, HAVING, COALESCE, TOP N

## 🛠️ Tools
SQL Server · T-SQL
